SEO

  1. Met taggar hjälper googles robot-spider att indexera webbsidan med hjälp av nyckel ord.
  2. Skriv länkar på ett bra sätt så att det ska vara lätt för en viewer att hitta den.
  3. Sidan måste ha relevant ord för att google skulle kunna samla ihop så mycket innehåll så möjligt.

SEO och nyckelord

  1. Att bra metod är google och på liknade innehåll som är hög kvalitet.
  2. Man kan även kolla på google trends var det är som trendjar.

Nätverk

  1. Webbplatsen skickar en request till en server från webbsidan som vi tittar efter.
  2. Webserver responds och skickar tillbaka the request query.

Responsiv webbdesign

Responsiv webbdesign är ett breakthrough teknologi som genomförds med CSS3.
Det betyder att sidor innan CSS3 2014 brukade inte att anpassa deras innehåll om en skärmen hade annorlunda storlek.
Man kan använda tex media queries för att anpassa innehålet på en lämplig storlek.
Programmeraren sätter några parameter typ "@media screen and (max-width: 900px) {...}
"Det betyder är skärmen mindre än 900px så gör följade instruktioner.

Review
